How to run?

0. Navitage your terminal to the root folder \space-invaders

1. You may want to create a vertual environment to install necessary libraries:
In the root folder run:
    python -m venv .venv

2. You now want to activate it:
    source .venv/bin/activate

3. Now install all libraries (make sure you have pip installed already):
    pip install -r requirements.txt

4. In the root folder you can run the game by 
    python game.py

5. To test the game depending on your machine you may try either 
    python -m pytest
   or 
    pytest

6. To see the test coverage:
    python -m pytest --cov 
   or 
    pytest --cov
